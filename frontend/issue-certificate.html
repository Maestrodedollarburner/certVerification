Issuance Process
  1. Select a registered student
  2. Enter the degree to be awarded
  3. System generates unique Certificate ID
  4. SHA-256 hash computed from certificate data
  5. Hash recorded on Ethereum blockchain
  6. QR code generated for verification
  7. PDF certificate available for download
